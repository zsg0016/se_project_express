# WTWR (What to Wear?): Back End

- This project is the back-end for the WTWR (What to Wear) project. User information and their clothing items can be added to a database.

## Technologies Used

- Node.js - JavaScript runtime environment
- Express.js - Web application framework for building REST APIs
- MongoDB - NoSQL database for storing user and clothing item data
- Mongoose - MongoDB object modeling library with schema validation
- Validator - Library for string validation (URLs, emails, etc.)
- ESLint with Airbnb Config - Code linting and style enforcement
- Nodemon - Development tool for automatic server restarts
- Prettier - Code formatter for consistent styling

## Running the Project

`npm run start` — to launch the server

`npm run dev` — to launch the server with the hot reload feature
